<h1>Barber Shop</h1>

<p>
    A modern, responsive website for a barbershop built with HTML, CSS, and JavaScript. 
    This project features a clean design with smooth animations, a functional slider, 
    booking modal, and mobile-friendly navigation.
</p>
<h2>📋 <span>Features</span></h2>

<ul>
  <li>
    <h3>Responsive Design</h3>
    <p>Fully responsive layout that works on mobile, tablet, and desktop devices</p>
  </li>
  <li>
    <h3>Image Slider</h3>
    <p>Automatic and manual image slider with pagination dots and navigation controls</p>
  </li>
  <li>
    <h3>Booking Modal</h3>
    <p>Pop-up booking form with form validation</p>
  </li>
  <li>
    <h3>Mobile Burger Menu</h3>
    <p>JavaScript-powered hamburger menu for mobile devices</p>
  </li>
  <li>
    <h3>Smooth Scrolling</h3>
    <p>Smooth scroll navigation to page sections</p>
  </li>
  <li>
    <h3>Interactive Elements</h3>
    <p>Hover effects, animations, and transitions</p>
  </li>
  <li>
    <h3>SVG Icons</h3>
    <p>Scalable vector icons for better performance</p>
  </li>
  <li>
    <h3>Contact Form</h3>
    <p>Footer booking form with animated labels</p>
  </li>
  <li>
    <h3>Services Section</h3>
    <p>Price list with two-column layout on desktop</p>
  </li>
  <li>
    <h3>Team Section</h3>
    <p>Barber profiles with social media links</p>
  </li>
  <li>
    <h3>Photo Gallery</h3>
    <p>Responsive image grid</p>
  </li>  
</ul>

<h2>🛠️ <span>Technologies Used</span></h2>
<ul>
    <li>HTML5
    <p>Semantic markup structure</p>
    </li>    
    <li>CSS3
    <p>Flexbox, Grid, animations, media queries</p>
    </li>    
    <li>JavaScript
    <p>DOM manipulation, event handling, form validation</p>
    </li>
    <li>Flexbox</li>
    <li>CSS Grid</li>
    <li>Google Fonts
    <p>Raleway and Open Sans fonts</p>
    </li>
    <li>SVG Sprites
    <p>Icon system for scalability</p>
    </li>
</ul>
